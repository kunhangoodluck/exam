# 📊 Scratch 期末測驗系統 - 專案總覽

## 🎯 系統功能

### 學生端功能
1. **登入系統**
   - 輸入姓名、班級、座號
   - 自動驗證必填欄位

2. **程式知識測驗**
   - 24 題 Scratch 程式概念選擇題
   - 即時進度顯示
   - 可前後切換題目
   - 自動計分系統

3. **中文打字測驗**
   - 8 分鐘倒數計時
   - 即時統計：
     * 打字速度（字/分鐘）
     * 正確率（%）
     * 總字數
     * 錯誤次數
   - 打字內容：王維《鹿柴》詩

4. **排行榜**
   - 班級排行榜（可篩選班級）
   - 校園總排行榜
   - 打字速度排行榜
   - 即時更新

### 教師端功能（管理後台）
1. **統計儀表板**
   - 參與人數統計
   - 平均測驗分數
   - 平均打字速度
   - 平均正確率

2. **資料管理**
   - 查看所有學生成績
   - 匯出成績 CSV
   - 匯出成績 JSON
   - 刪除單筆資料
   - 清空所有資料

## 📈 評分機制

### 程式知識測驗
- 總分：100 分
- 每題約 4.17 分
- 答對得分，答錯不扣分

### 打字測驗
- 主要指標：速度（字/分鐘）
- 次要指標：正確率（%）
- 參考標準：
  * 優秀：50+ 字/分鐘
  * 良好：35-50 字/分鐘
  * 及格：20-35 字/分鐘

### 綜合排名
```javascript
總分 = 測驗分數 + (打字速度 / 2)
例如：85 分 + (45 / 2) = 107.5 分
```
